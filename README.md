# Robotic Arm with Voice and Hand Gesture Control

Robotic arm controlled using voice commands and hand gestures, enabling intuitive human-robot interaction.

---

## Features

- Voice-based control of the robotic arm  
- Home Appliance control using voice command 
- Face Tracking 
- Hand Gesture control

---

## Software and Hardware Used

- Python  
- OpenCV  
- ESP32  
- Embedded C  
- Servo Motors  

---

## Project Structure

- `code/` → Python (gesture/vision) and ESP32 control code  
- `hardware/` → Components and setup details  
- `demo/` → Images and video demonstration  

---

## System Overview

- Laptop (Python/OpenCV) processes hand gestures and face tracking, and sends control commands to the ESP32.  
- ESP32 receives commands from the laptop and voice module, and controls the servo motors accordingly.
